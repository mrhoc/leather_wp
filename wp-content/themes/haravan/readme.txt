== Description ==

Haravan is a clean and modern WordPress theme designed for blogs and business websites. It features a responsive design that works perfectly on all devices, customizable colors through the WordPress Customizer, and support for featured images.

Key Features:
* Responsive design
* Customizable colors
* Featured image support
* Widget-ready sidebar
* SEO optimized
* Translation ready
* Custom logo support
* Social media integration ready

== Installation ==

1. Upload the theme files to the `/wp-content/themes/haravan` directory, or install the theme through the WordPress admin panel.
2. Activate the theme through the 'Themes' menu in WordPress.
3. Customize the theme through the WordPress Customizer.

== Frequently Asked Questions ==

= Does this theme support custom logos? =

Yes, you can upload a custom logo through the WordPress Customizer.

= Is this theme translation ready? =

Yes, the theme is fully translation ready and includes .pot files for translation.

= Does this theme support featured images? =

Yes, the theme supports featured images for posts and pages.
